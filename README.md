# Friend.Tech on Twitter

This is a chrome extension that adds visiblity of friend.tech into x/twitter.com

- Adds supply and price (usd/eth) onto profiles
- Marks those on friend.tech as FT blue on your timeline.

## Installation

1. Clone this repo to your machine
2. Open [chrome://extensions/](chrome://extensions/) in your browser
3. Check the box for Developer mode in the top right
4. Press **Load Unpacked** button and select the "extensions" folder with the code
